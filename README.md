## 📲 วิธีติดตั้ง (Add to Home Screen)

### iOS (Safari)
1. เปิด Safari → ไปที่ Demo link
2. กดปุ่ม **Share** (⬆️)
3. เลือก **"Add to Home Screen"**

### Android (Chrome)
1. เปิด Chrome → ไปที่ Demo link
2. กดปุ่ม **⋮** → **"Add to Home screen"**

### Desktop (Chrome/Edge)
1. เปิด Browser → ไปที่ Demo link
2. กดไอคอน **Install** (⊕) ที่ Address Bar
